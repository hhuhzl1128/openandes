## Resource Inventory
### ANDES built-in cases
Representative systems include IEEE, Kundur, WECC, NPCC, WSCC and others, with mixed file formats and rich event/controller examples.

### Demo repository
Contains scenario-focused examples (oscillation, frequency response, governor studies), system-scale examples (for example texas7k/hawaii), and workflow utilities.

## Use Strategy
- Start from built-in stable cases
- Reuse demo patterns for scenario design
- Keep local mapping between case purpose and required dependencies
