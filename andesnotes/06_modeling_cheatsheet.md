# ANDES Modeling Cheat Sheet (Minimal Custom Model)

## Purpose
Quick reference for execution and validation during minimal custom-model implementation.

## Principles
- Make it run first, then optimize
- Every step must have a verifiable command/output

## Fast Checklist
1. Environment readiness
2. Model copy/rename
3. Registry update
4. Prepare execution
5. Import and runtime verification
6. Regression run

## Typical Failure Points
- Missing registration
- Incorrect model path
- Inconsistent variable/equation definitions
- Initialization issues after successful parsing
