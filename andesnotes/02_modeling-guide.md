# ANDES Study Notes — Modeling Guide

Source: modeling/index.html

## Purpose
The Modeling Guide is developer-facing documentation for understanding and extending ANDES models.

## Major Sections
1. Modeling Concepts
2. Model Components
3. Creating Models
4. Testing and validation workflow

## Core Position
ANDES uses a symbolic-first modeling style. Users define model structures and equations, and the framework handles derivative generation, code preparation, and numerical execution.
