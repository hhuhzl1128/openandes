## 07 Eigenvalue Analysis
Workflow:
1. Build operating point
2. Linearize the system
3. Compute eigenvalues and mode statistics

Focus:
- Damping and oscillatory modes
- Stability interpretation with engineering context

## 08 Parameter Sweeps
- Automate repeated runs over parameter grids
- Store outputs with unique run IDs
- Compare trends, not isolated points

## 09 Contingency Analysis
- Define events (line/generator/load)
- Evaluate system response robustness
- Capture critical boundaries and failure modes

## 10 Dynamic Control
- Modify setpoints/events during simulation
- Validate control response timing and magnitude

## 11 Frequency Response
- Analyze nadir, RoCoF, settling behavior
- Compare scenarios under standardized disturbances

## Inspecting Model Equations
- Verify variable definitions and equation mapping
- Check consistency between model equations and expected physical behavior
