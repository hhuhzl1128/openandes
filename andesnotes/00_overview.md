# ANDES Study Notes — Overview

Source: https://docs.andes.app/en/latest/

## Project Positioning
ANDES is an open-source Python framework for power-system modeling and numerical analysis.

Core capabilities:
- Power flow analysis
- Time-domain simulation (transient stability)
- Small-signal stability (eigenvalue analysis)
- Hybrid symbolic–numeric modeling workflow
- Support for modern renewable-generation models

## Documentation Topology
Main sections:
- Tutorials
- Modeling Guide
- Reference
- Verification

## Practical Learning Goal
Build a reproducible workflow from case loading to dynamic simulation, then extend to model customization and validation.
