# ANDES Practice — Minimal Runnable Custom Model

## Objective
Build a fully reproducible minimum workflow for creating and validating a custom model.

## Workflow
1. Prepare environment
2. Copy a simple official static model as baseline
3. Rename and register custom model
4. Run prepare/code-generation
5. Verify model discovery and import
6. Execute a minimal regression run

## Validation Checklist
- Environment works
- Model registration is effective
- Generated code is available
- Runtime does not fail at parse/init stage

## Troubleshooting Path
- Check registration and import paths first
- Check parameter/variable declarations
- Check equation compatibility and initialization order
